# reload

Reads the current configuration file (again) from the file system and parses its contents.

:::info

All configurations can be found at `~/.local/share/veiled/configs/<process.exe>/[NAME].conf`.

:::

## Declaration

`.reload()`

## Syntax

```lua
Configs.reload()
```

## Return value

The function returns no value.
